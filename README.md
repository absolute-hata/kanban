# 📋 项目看板

单文件项目看板工具，支持周视图、任务管理、暗色主题。纯 HTML + CSS + JS，零依赖。

![项目看板](https://img.shields.io/badge/技术栈-HTML%2FCSS%2FJS-blue) ![单文件](https://img.shields.io/badge/大小-104KB-brightgreen)

## 功能

- **周日历视图** — 按周查看任务，左右切换周
- **项目分组** — 左侧栏管理多个项目，每个项目独立看板
- **任务卡片** — 优先级（高/中/低）、预计时间、计划日期、类别标签
- **状态管理** — 未完成/已逾期/进行中/已完成，颜色区分一目了然
- **左滑删除** — 移动端触摸滑动删除（`touchstart`/`touchmove`/`touchend`）
- **搜索** — 全局搜索任务标题
- **暗色主题** — 一键切换，CSS 变量控制
- **弹窗编辑** — 点击卡片编辑详情，支持模块标签添加
- **云存储** — 支持坚果云 WebDAV 同步（可选）
- **GSAP 动画** — 弹窗缩放弹入、周切换卡片 stagger 滑入、完成打勾弹性缩放

## 技术要点

| 项目 | 实现 |
|------|------|
| 布局 | CSS Flexbox + Grid，左侧栏固定 + 主区域自适应 |
| 响应式 | 移动端适配，侧栏自动收起 |
| 动画 | GSAP `fromTo` / `to` + `stagger` |
| 数据 | 本地 JSON 对象 + localStorage / 云 API |
| 暗色模式 | CSS 变量 `:root` / `body.dark` 切换 |
| 任务颜色 | 逾期(红)、延迟(橙)、准时(绿)、完成(灰) |
| 搜索 | 输入即时过滤，点击结果跳转到对应周 |

## 使用

直接用浏览器打开 `项目看板.html`：

```bash
# macOS
open 项目看板.html

# 或本地起服务（可选）
npx serve .
```

## 开发背景

为团队周会设计的轻量看板，核心需求：
- 按周规划任务，一眼看到谁做了什么、什么时候做的
- 优先级和逾期状态要醒目
- 手机上也要能用（左滑删除、侧栏收起）
- 不想装任何依赖，一个 HTML 文件带走
